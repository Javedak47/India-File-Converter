# India-File-Converter

✨ **Features:**
- 📝 PDF to Text Conversion
- 🖼️ PDF to JPG Conversion (Coming Soon)
- 🏞️ Extract Images from PDF (Coming Soon)
- 🔒 Secure & Private Processing
- ⚡ Progress Tracking
- 📱 Mobile Friendly
- No File Size Limits (50MB Recommended)

## 🔧 Technology

- **Pure Client-Side Processing**: All conversions happen in your browser—no data is sent to any server
- **PDF.js Library**: Powered by Mozilla's robust PDF processing library
- **Responsive Design**: Works seamlessly on Mobile, Tablet, and Desktop

## 🌍 Language

- **Hindi Interface** (हिंदी इंटरफेस)
- Made with ❤️ for Indian Users

## 🚀 Features

### ✅ Currently Implemented
- **PDF to Text**: Extract searchable text from PDF documents
- **File Validation**: Automatic checks for file type and size (max 50MB)
- **Progress Tracking**: Real-time progress bar during conversion
- **Instant Download**: Download converted files with one click
- **Responsive UI**: Beautiful gradient design with smooth animations

### 🔄 Coming Soon
- PDF to JPG image conversion
- Extract images from PDF documents
- Batch file processing
- Support for scanned PDFs (OCR)

## 📖 How to Use

1. **Select a PDF File**: Click "फ़ाइल चुनें" (Choose File) button
2. **Choose Conversion Type**: Select from available conversion options
3. **Start Conversion**: Click "कन्वर्ट करें" (Convert) button
4. **Download Result**: Once complete, download your converted file

## 🔒 Privacy & Security

- ✅ All processing happens locally in your browser
- ✅ Your files are never uploaded to any server
- ✅ 100% private and secure
- ✅ Works offline (after initial page load)

## 📱 Supported Devices

- ✅ Desktop browsers (Chrome, Firefox, Safari, Edge)
- ✅ Mobile browsers (iOS Safari, Chrome Mobile)
- ✅ Tablet browsers
